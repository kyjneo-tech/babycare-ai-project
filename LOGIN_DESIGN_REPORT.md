## 🎨 디자인 시스템

### 색상 팔레트
```css
/* 배경 그라데이션 */
from-pink-100 via-purple-100 to-blue-100

/* 메인 그라데이션 (버튼, 제목) */
from-pink-500 to-purple-600

/* 포커스 색상 */
border-purple-400
ring-purple-100

/* 게스트 버튼 */
from-blue-400 to-cyan-400
```

### 주요 스타일 요소

#### 1. **그라데이션 텍스트**
```tsx
className="bg-gradient-to-r from-pink-500 via-purple-500 to-blue-500 bg-clip-text text-transparent"
```

#### 2. **입력 필드**
```tsx
className="w-full px-4 py-3 border-2 border-purple-200 rounded-xl 
           focus:border-purple-400 focus:ring-4 focus:ring-purple-100 
           transition-all outline-none"
```

#### 3. **메인 버튼**
```tsx
className="w-full py-3 px-4 bg-gradient-to-r from-pink-500 to-purple-600 
           text-white font-bold rounded-xl shadow-lg hover:shadow-xl 
           hover:from-pink-600 hover:to-purple-700 
           transform hover:scale-105 transition-all duration-300"
```

#### 4. **카드 컨테이너**
```tsx
className="bg-white p-8 md:p-10 rounded-3xl shadow-2xl 
           transform hover:scale-105 transition-transform duration-300"
```

## 🎭 애니메이션 효과

### 배경 장식 아이콘
```tsx
// 로그인 페이지
🍼 - animate-bounce (top-left)
👶 - animate-pulse (top-right)
🧸 - animate-bounce delay-100 (bottom-left)
⭐ - animate-pulse delay-200 (bottom-right)

// 회원가입 페이지
🎈 - animate-bounce (top-left)
🌟 - animate-pulse (top-right)
🎨 - animate-bounce delay-100 (bottom-left)
🌈 - animate-pulse delay-200 (bottom-right)
```

### 인터랙션 애니메이션
- **카드 hover**: `transform hover:scale-105`
- **버튼 hover**: `hover:scale-105 + shadow 증가`
- **입력 필드 focus**: `ring-4 ring-purple-100`
- **에러 메시지**: `animate-shake`
- **로딩 스피너**: `animate-spin`
