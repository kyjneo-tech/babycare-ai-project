## 개발 원칙 (TDD & SOLID)
- TDD 사이클 준수: Red → Green → Refactor
- Tidy First: 구조 변경(S)과 행동 변경(B) 분리
- SOLID 원칙 준수 (특히 SRP, OCP, DIP)
- 데이터베이스 3NF 준수
- shadcn/ui 기반 재사용 가능한 컴포넌트
- 기능 구현 후 npm run test && npm run build 실행
- 빌드 성공 시 커밋 후 푸쉬

- 긍정적 메시지로 디버깅 
- 중요 개념은 비유로 설명
